PlayStation 1 Emulator

A cycle-accurate MIPS R3000A emulator with advanced graphics features. Built with C++, OpenGL, and ImGui.

Key Features

  • Cycle-accurate MIPS R3000A CPU emulation
  • Advanced GPU emulation with texture support
  • Post-processing shaders: upscaling, bloom, and CRT effects
  • CRT effects include scanlines, noise, and edge darkness
  • ImGui-based debug interface for system inspection
